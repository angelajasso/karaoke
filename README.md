# 🎤 Karaoke Dashboard

## Overview
Interactive karaoke-style web application that synchronizes audio playback with timed lyrics.

## Features
- Upload audio files
- Load synchronized lyrics (.lrc)
- Real-time lyric highlighting
- Playback controls
- Progress tracking

## Tech Stack
- HTML
- CSS
- JavaScript (ES Modules)
- Web Audio API (future)

## Architecture
Modular structure:
- state management
- audio engine
- lyrics parser
- UI rendering
- event system

## Goal
Build an interactive audio-reactive experience for musicians and learners.
